📄 README — Aplicação IPMaster
Nome da aplicação: IPMaster
Versão: 1.0.0
Autor: Gabriel Ferreira e Douglas Santos
Descrição: Aplicação para conversão de números entre diferentes bases numéricas.

🧾 O que faz a aplicação?
A IPMaster permite converter números entre as seguintes bases:

  - Binário (base 2)
  - Octal (base 8)
  - Decimal (base 10)
  - Hexadecimal (base 16)

A aplicação valida automaticamente os valores introduzidos de acordo com a base de origem e mostra o resultado da conversão, além de manter um histórico das conversões anteriores.


🧰 Funcionalidades
Conversão instantânea entre bases
Validação automática de input
Interface moderna
Histórico das últimas conversões
Botão para limpar os campos e o histórico
Design adaptado para computador ou dispositivos móveis


🛠 Como utilizar
Introduz um número no campo de texto.
Seleciona a base de origem e a base de destino através dos menus suspensos.
Clica no botão “Converter”.
O resultado será apresentado num cartão, e a conversão será registada no histórico.
Para limpar os campos e o histórico, usa o ícone de “refresh” no canto superior direito.


📂 Estrutura geral
Campo de introdução de número
Dropdowns: seleção de base de origem e de destino
Botão de conversão
Resultado em destaque
Histórico das conversões mais recentes

📌 Notas
A conversão só funciona com números válidos de acordo com a base selecionada.
O histórico é temporário e será perdido ao limpar ou reiniciar a aplicação.
